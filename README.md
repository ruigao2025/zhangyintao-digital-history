# 张隐韬烈士数字历史纪念网站

基于《张隐韬烈士日记》及相关史料构建的数字化纪念与研究网站。

## 网站内容

- **首页** — 生平传记、核心论点、数字概览、阅读路径引导
- **思想分析** — 四阶段思想演变脉络 + 三篇专题深度分析
- **人物网络** — 32位交往人物关系图谱与结构分析
- **事件对照** — 8组关键事件三栏交叉审视 + 五则新材料互证
- **日记全文** — 1923.8—1924.9 日记全文，含关键条目批注
- **史料研究** — 原始史料、新发现材料、学术研究汇总

## 资料来源

- 河北省南皮县档案馆《张隐韬烈士日记》连载一至五
- 张洪涛《关于张隐韬生平事迹的几则新材料》
- 石慧《一名共产党员学生眼中的黄埔军校》

## 技术说明

纯静态网站（HTML/CSS/JS），无需服务器，可直接打开 index.html 浏览。
